Товары за $0.01 рассчитаны на новых пользователей, поэтому чтобы не закрывали заказы нужно принимать меры предосторожности. Вот самый актуальный способ заказывать товары:

  • Скачиваем портативную версию браузера Firefox и устанавливаем расширения:
    Canvas Defender - для генерации случайных отпечатков браузера
    Disable WebRTC - отключает WebRTC (чтобы не утек Ваш реальный IP адрес)
    Random User-Agent - для генерации случайного User-Agent
    Если будете использовать прокси, то установите еще Spoof Timezone и FoxyProxy Standard
  • В настройках "Приватность и защита" поставьте галочку "Удалять куки и данные сайтов при закрытии Firefox"
При каждом открытии браузера нажимайте на расширение Canvas Defender, там нажмите кнопку Genegate new noise, в расширении Random User-Agent нажмите кнопку "Сгененировать новый", для смены IP можно использовать прокси/ВПН или 3g телефона (При каждом включении мобильного интернета у вас будет новый IP). Заказы оплачивайте только через WebMoney чтобы AliExpress не мог Вас отследить.

Адрес доставки нужно коверкать чтобы AliExpress ничего не заподозрил, номер указывать рандомный либо коверкать (например 00-0000/00-0-00).
Вот основные причины почему закрывает заказы:
  • Плохая коверка адреса/номера телефона (рекомендуем использовать рандомный номер телефона)
  • IP адрес уже использовался. Такое часто бывает если использовать публичные прокси/ВПН, рекомендуем использовать мобильный интернет
  • Товар заблокирован. Есть товары из-за которых всегда закрывает заказы, поэтому заказывайте всегда разные товары.
